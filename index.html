Pillars of Excellence

Project Pisces is built on a foundation of security, automation, and deep observability, empowering teams to build and innovate with speed and confidence.

Secure by Design

Automated guardrails and proactive threat detection are built-in. With centralized security services like GuardDuty, Security Hub, and Wiz, the platform meets the highest standards for compliance and data protection from day one.

Automated Operations

Leveraging Infrastructure as Code (IaC) with Terraform and a robust CI/CD pipeline via GitHub Actions. The Account Factory for Terraform (AFT) enables fully automated, consistent, and rapid provisioning of new AWS accounts.

Centralized & Observable

Gain complete visibility across the entire environment. Centralized logging and monitoring through Splunk and Datadog, combined with unified identity management via AWS Identity Center, provide a single pane of glass for all operations.
