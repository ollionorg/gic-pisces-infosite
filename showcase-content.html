Pillars of Excellence

Project Pisces is built on a foundation of security, automation, and deep observability, empowering teams to build and innovate with speed and confidence.

Secure by Design

Leveraging AWS Control Tower for foundational security, Project Pisces enforces preventive and detective controls. With centralized services like AWS GuardDuty, Security Hub, and Wiz, the platform meets the highest standards for compliance. Integration with CyberArk ensures robust secret management.

Automated Operations

Utilizing Account Factory for Terraform (AFT) and a robust CI/CD pipeline, the platform enables automated provisioning of AWS accounts. The integration with BMC Helix and CloudCell provides a library of approved, deployable AWS resources, transforming account creation into a seamless workflow.

Centralized & Observable

Gain complete visibility with centralized logging and monitoring through Splunk and Datadog. AWS Config provides continuous monitoring of resource configurations, ensuring compliance. This unified approach offers a single pane of glass for all operations and security monitoring.
